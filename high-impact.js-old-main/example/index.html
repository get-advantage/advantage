High Impact JS
Template Demos

An open-source JavaScript library for creating high-impact advertising experiences. Explore our templates and build amazing ad units for your web content.

  • Topscroll

    A premium ad format that appears at the top of the page and responds to scroll behavior. Perfect for creating impactful above-the-fold advertising experiences.

    View Topscroll Demo →

  • Midscroll

    An engaging ad format that appears while scrolling through content. Ideal for maintaining user attention mid-article or mid-content.

    View Midscroll Demo →

  • Double Fullscreen

    An immersive advertising experience that utilizes the full screen real estate for maximum impact, with support for multiple creative elements.

    View Double Fullscreen Demo →

  • Sitebar

    A responsive side-aligned ad format that stays in view while scrolling. Great for persistent brand visibility without disrupting the main content.

    View Sitebar Demo →

  • Skins

    A customizable template that allows for site skinning and background takeovers. Perfect for creating immersive branded experiences.

    View Skins Demo →

  • Takeover

    A full-page takeover format that creates an immersive brand experience. Features a customizable top bar where you can add custom HTML content with text and styling.

    View Takeover Demo →

  • Xandr Integration

    Demonstrates integration with Xandr (formerly AppNexus) ad platform, showing how to implement programmatic advertising with High Impact JS.
